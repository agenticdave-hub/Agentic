# Angličtina hrou

Webová aplikace pro děti 6–9 let – procvičování angličtiny hravou formou.

## Soubory
- index.html – Landing page
- game.html – Hravý kvíz
- pro-rodice.html – Sekce pro rodiče

## Spuštění
Soubor `index.html` nebo `game.html` stačí otevřít v prohlížeči. Ideální pro nasazení na GitHub Pages.
